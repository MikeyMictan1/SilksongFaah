# SilksongFaah
A lightweight Hollow Knight: Silksong mod that plays the "Faah" sound effect when Hornet dies. Why did I make this? I'm still looking for that answer...

<img width="256" height="256" alt="icon" src="https://github.com/user-attachments/assets/12815dcd-57a3-4db8-b876-2eb82d0a0b9c" />

## Features
- Plays "Faah" on hornet death. That's it.
- Easy to install through Thunderstore and Gale Mod Manager.

## Install
**If installing manually:** Create and place the contents into a `SilksongFaah` folder in your games `BepInEx/plugins` folder (requires the BepInEx modloader).
