# SilksongFaah
A lightweight Hollow Knight: Silksong mod that plays the "Faah" sound effect when Hornet dies. Why did I make this? I'm still looking for that answer...

## Features
- Plays "Faah" on hornet death. That's it.
- Easy to install through Thunderstore.

## Install
**Thunderstore:** [install from thunderstore here.](PASTE_THUNDERSTORE_INSTALL_URL_HERE)

**If installing manually:** place the `SilksongFaah` folder in `SilksingFaah-1.0.0/BepInEx/plugins/` into your games `BepInEx/plugins` folder.
